ckanext-admin
=============

This extension provides administrative tools for CKAN. The extension is 
created as a side project for ckanext-kata, see
http://github.com/kata-csc/.

This Extension supports CKAN version 2.0.2.

Installation
============

Install extension with

  pip install -e git://github.com/pkoskinen/ckanext-admin.git#egg=ckanext-admin
  
Current status
==============

Currently the extension only provides reporting.
